# pygame-first-course

Learning the basics of game development with Pygame

## Setup

Navigate to the `shooter_assets` folder and follow the steps for your OS:

### Linux/macOS

```bash
cd shooter_assets
python3 -m venv venv
source venv/bin/activate
pip install -r ../requirements.txt
```

### Windows

```bash
cd shooter_assets
python -m venv venv
venv\Scripts\activate
pip install -r ..\requirements.txt
```

## Running the Game

Once the setup is complete, run:

```bash
python3 shooter.py
```
